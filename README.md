🛒 Blinkit Sales Analysis Dashboard (Power BI)
📌 Project Overview

This project presents an interactive Blinkit Sales Analysis Dashboard built using Power BI. The dashboard provides insights into sales performance, product distribution, outlet characteristics, and customer ratings to support data-driven decision-making.

🎯 Objective

The main goal of this project is to analyze Blinkit's retail data and uncover meaningful insights such as:

Total sales performance

Product category contribution

Outlet-wise analysis

Customer satisfaction trends

📊 Key Insights

💰 Total Sales: $1.20M+ revenue generated

⭐ Average Rating: 3.9 across products

📦 Total Items Sold: 9K+

🏪 Outlet Performance: Tier 3 outlets contribute the highest sales

🥗 Product Trends: Fruits, Snacks, and Household items lead in sales

📈 Sales Trend: Peak sales observed around 2018

📌 Dashboard Features

Interactive Filters Panel

Outlet Location Type

Outlet Size

Item Type

Visualizations Included:

KPI Cards (Total Sales, Avg Sales, Rating, Items)

Sales Trend by Outlet Establishment Year

Donut Charts (Fat Content, Outlet Size)

Bar Charts (Item Type Sales, Outlet Comparison)

Outlet Type Performance Table

🛠️ Tools & Technologies

Power BI – Data Visualization

Excel / CSV – Data Source

DAX – Data Modeling & Calculations

📁 Project Structure

Blinkit_Dashboard.pbix → Power BI Dashboard File

dataset.csv → Dataset used for analysis

README.md → Project Documentation

🚀 How to Use

Download the .pbix file from this repository

Open it in Power BI Desktop

Interact with filters and visuals to explore insights

📌 Use Case

This dashboard can help:

Business analysts understand sales trends

Retail managers optimize inventory

Companies improve product strategy
